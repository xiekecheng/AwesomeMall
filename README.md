# Vue 3 + Vite
本项目采用Vue3+Vite开发

```
npm init vite@latest
```

```
// 引入 vue-router@4
npm install vue-router@4
```

```
// 引入 elementPlus
npm install element-plus --save
```

```
// 引入vuex
npm install vuex@next --save
```

引入sass

```
# .scss and .sass
npm install -D sass
```

引入normalize.css初始化浏览器样式

```
npm install normalize.css
```
